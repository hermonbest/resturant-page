## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/resturant-page.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd resturant-page
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

## üíª Usage

To start the development server, run the following command:

```bash
npm start
```

This will open the website in your default browser at `http://localhost:8080`.

The website has the following sections:

*   **Home:** The landing page with a welcome message and a booking section.
*   **Menu:** Displays the restaurant's menu.
*   **About:** Provides information about the restaurant.
*   **Booking:** Allows users to book a table.
